# 关于 TierSum

## 产品概述

TierSum 是一个**分层摘要知识库**——通过多层抽象和热/冷文档分层保留文档结构、以章节为检索粒度的无需 RAG 的文档检索系统。

## 为何选择 TierSum

常见检索系统把文档切碎成小块做相似度匹配，结构模糊、语境丢失。TierSum 保持**清晰的层级结构**——文档概述、章节摘要与原文——并配合**标签与主题**，让你像浏览目录一样导航知识，而非靠向量猜测碎片边界。

冷热两档均以章节为界，而非固定尺寸碎片。每次检索返回**有意义的完整章节**。

## 能做什么

- **搜索** — 渐进式查询沿**标签 → 文档 → 章节**逐步收窄，像先扫目录再翻到对应小节
- **文档** — 导入 Markdown 内容。热文档按章节获得完整 LLM 分析；冷文档同样以章节粒度建索引与检索
- **标签** — 浏览自动生成的目录标签，按 LLM 策展的主题分组，随库增长保持导航意义
- **集成** — 完整 REST API + MCP 协议，支持 AI 代理与自动化工作流

## 适合谁

以 Markdown 为中心的知识团队：内部手册、架构说明、运维笔记、研究纪要，以及需要程序化接口对接的智能体与工作流。浏览器界面与程序接口共用同一套数据，减少知识孤岛。
